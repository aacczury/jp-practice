# 日本語 PWA — deploy to GitHub Pages

This folder is a self-contained static web app **and its own git repo** (separate from the life repo,
so the audio lives with the app). The lesson data/audio are generated by `../jp-web-build.py`.

## First-time hosting
```
gh auth login                       # one-time, browser login
cd ~/Documents/life/jp/web
gh repo create jp-practice --public --source=. --push
gh api -X POST repos/$(gh api user -q .login)/jp-practice/pages --input - <<<'{"source":{"branch":"main","path":"/"}}'
```
Then open **https://<your-username>.github.io/jp-practice/** on your phone →
Share → **Add to Home Screen**. (First load needs internet; after that it works offline.)

## Update after adding lessons (e.g. new days)
```
cd ~/Documents/life/jp
python3 jp-web-build.py              # regenerate data + audio + service worker
cd web && git add -A && git commit -m "update lessons" && git push
```
The service-worker cache name is content-hashed, so phones pick up changes on next open.
